# Contributing to AstraSRE

We welcome community contributions! Please follow these guidelines to help keep the project clean, secure, and robust.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct.

## Getting Started

1. **Fork the Repository**: Create a fork of AstraSRE on your own GitHub account.
2. **Clone Locally**: Clone your fork to your local workstation.
3. **Set Up telemetry**: Follow the instructions in the `README.md` to start the local Docker Compose stack.
4. **Make Your Changes**: Keep commits small, atomic, and well-described.

## Pull Request Guidelines

* **Staging**: Always submit pull requests against the `main` branch.
* **Testing**: Ensure any code changes pass local validation and build steps without errors.
* **Documentation**: Update corresponding markdown files if introducing new endpoints or changing configuration parameters.
* **Style**: Align with Python's PEP 8 guidelines for backend logic and Tailwind conventions for frontend components.
