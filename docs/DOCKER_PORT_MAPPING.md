# Docker Port Configurations

Port maps for local AstraSRE orchestration:
- Frontend gateway: port `5000` (metrics `8003`).
- Order handler: port `5001` (metrics `8001`).
- Payment handler: port `5002` (metrics `8000`).
- Inventory handler: port `5003` (metrics `8002`).
- Notification handler: port `5004` (metrics `8004`).
